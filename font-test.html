Notes from Monday standup

Server-side sync is hash-based: the client sends its local state, the server responds with what's different. See Sync protocol notes for the full spec.

"The filename is the title. No case changes, no dash-to-space, no transformations." — design doc

This means grocery list.md becomes the title "grocery list" exactly as-is. The sanitizeFilename() function only strips characters that would break the filesystem.
